# ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/yourusername/parth-tech-website.git
```

### 2️⃣ Navigate to project directory

```bash
cd parth-tech-website
```

### 3️⃣ Run backend services

Java Backend:

```bash
cd backend/java-api
mvn spring-boot:run
```

Python AI Services:

```bash
cd backend/python-ai-services
python app.py
```

### 4️⃣ Open frontend

Open the HTML files or run a development server.
